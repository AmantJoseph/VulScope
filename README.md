
# VulScope CLI Tool

## Features
- Search CVEs by keyword or CVE ID
- Severity filtering
- PDF + JSON export
- Error handling (network, API, input)

## Usage
python main.py --keyword nginx
python main.py --cve-id CVE-2021-44228
python main.py --keyword nginx --severity HIGH --pdf --json
